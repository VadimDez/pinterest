# Pinterest

Pinterest clone, part of FreeCodeCamp projects.

[Demo page](http://thepinterest.herokuapp.com/)

### Run

Install dependencies

```
npm i && bower i
```

then

```
grunt serve
```

#### Configuration

You can find configurations under

```
/server/config/enviroment
```

### Uses

- NodeJS
- AngularJS
- SCSS
- Karma
- Protractor
- Yeoman
